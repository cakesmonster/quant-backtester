# 日晷 (Sundial) — Open Design 提示词

请为 A 股市场看板「日晷」生成完整的前端设计。科技感暗色主题，风格参考 GitHub Dark / Bloomberg Terminal 的现代版。中文界面。所有数据目前是 mock 占位，后续会接入真实 API。

## 全局布局

- 顶部导航栏：Logo「日晷 Sundial」+ 5 个标签页切换（每日复盘 | 热榜 | 个股分析 | 策略回测 | 模拟账户）
- 底部状态栏：数据更新时间
- 配色：深蓝黑底色 (#0a0e17)，卡片 (#111827)，强调色用青绿 (#00d4aa) 表示涨、红色 (#f43f5e) 表示跌，文字灰白
- 字体：系统等宽 + 中文字体，数字用 Tabular Nums 对齐

## 页面 1：每日复盘

### 1.1 顶部日期选择器 + 情绪仪表盘
- 日期选择器（默认今天，可回溯历史日期）
- 下方 6 个指标卡片一字排开：
  - 涨停家数（绿底） | 跌停家数（红底） | 炸板家数（黄底）
  - 炸板率（XX%，炸板/(涨停+炸板)） | 连板晋级率（N板→N+1板） | 昨日涨停今日均收益（+X.X%）

### 1.2 昨日涨停今日表现
- 左侧卡片：水平条形图（四行）
  - 继续涨停 X 只 ████████
  - 红盘 X 只     ████████████████
  - 绿盘 X 只     ██████
  - 跌停 X 只     ██
- 右侧大字：平均涨幅 +2.3%（绿），最高/最低标注

### 1.3 竞价异动（仅当天显示）
- 历史日期隐藏此区块
- 表格：时间 | 代码 | 名称 | 类型（诱多 ⚠️ / 弱转强 🟢 / 爆量 📊）| 说明

### 1.4 连板天梯
- 主内容区，按连板数降序排列
- 每组：大号「N连板」标题 → 下面列出该组的所有股票
- 每只股票一行显示：代码 · 名称 · [板块标签] · 涨跌幅%（涨停=绿色加粗 +10.0%）
- **Hover 弹出信息卡**：
  ```
  ┌─────────────────────────┐
  │ 600xxx  某某股份          │
  │ 市值: 580亿  流通: 320亿  │
  │ 市盈率: 45.2              │
  │ 换手率: 18.7%  量比: 2.1  │
  │ 概念: 芯片, 半导体, AI   │
  └─────────────────────────┘
  ```
- **淘汰区**：连板天梯下方，灰色分隔线「── 淘汰 ──」，单列昨日涨停但今日破板的股票，灰色文字显示今日涨跌幅

### 1.5 板块攻击路线
- 底部区块，标题「板块资金流向」
- 左右两栏：🔥 今日领涨 | 📉 今日领跌
- 每栏列表项：板块名称 + 涨跌幅% + 水平进度条（涨=绿色，跌=红色）
- 中间一句话总结：切换趋势（如「资金从半导体撤离 → 涌向机器人、商业航天」）

## 页面 2：热榜

### 2.1 选择器
- 日期选择器 + 时段切换按钮组（11:30 | 15:00 | 21:00）
- 当前选中的时段按钮高亮

### 2.2 热榜表格
- 列：排名 | 代码 | 名称 | 热度值 | 概念标签 | 涨跌幅
- 排名前 3 名用金、银、铜色标记
- 涨停股整行绿色底色
- 每行可点击，展开下方子行显示分时小图（缩略版）+「查看详情 →」按钮跳转页面 3
- **Hover** 弹出信息卡（同页面 1 的信息卡样式）
- 点击涨停股的名称/行 → 跳转到页面 3（个股分析），并自动填入代码

### 2.3 找队友弹窗
- 点击涨停股右侧的「👥」按钮 → 弹出模态框
- 显示：同概念股票列表 + 走势相似股票列表（名称、相关性系数、当前涨幅）

## 页面 3：个股分析

### 3.1 输入区
- 顶部搜索框：输入股票代码（6 位数字），右侧「查询」按钮
- 查询后显示股票全称 + 所属板块标签
- 查询框下方提示：「仅支持当天数据」

### 3.2 布局（左右两栏）
**左栏（60%）：**
- 股票基本信息卡：市值 / 流通市值 / 市盈率 / 换手率 / 量比
- **分时走势图**（Plotly 交互图，占主要面积）：x 轴=时间（09:30-15:00），y 轴=价格，含均价线

**右栏（40%）：**
- **找队友** 卡片：
  - 「同概念」tab：列出同概念板块的股票（名称 + 涨跌幅）
  - 「走势相似」tab：列出分钟 K 线相关性 > 0.7 的股票（名称 + 相关系数 + 涨跌幅）
  - 每个队友可点击切换查看

### 3.3 大单异动
- 底部表格：时间 | 方向（买🟢/卖🔴）| 数量（手）| 金额（万）| 成交价
- 大单定义：≥ 3000 手

## 页面 4：策略回测

### 4.1 布局
- 策略选择下拉框（选中后下方显示策略描述）
- 参数：股票数量 / 窗口天数 / 每只次数 / 初始资金(万)
- ▶ 开始回测 按钮

### 4.2 结果区
- 顶部指标卡：平均收益 / 夏普比率 / 最大回撤 / 胜率 / 总交易数
- 中间：权益曲线叠加图（Plotly 交互，多个回测窗口的曲线半透明叠加）
- 底部表格：股票 | 窗口 | 收益率 | 夏普 | 最大回撤 | 胜率 | 交易次数

### 4.3 注意事项
- 此为嵌入/联动 quant-backtester，页面设计不需要实现具体回测逻辑，只需要 UI 框架
- 加一个「🔄 刷新策略」按钮用于热加载新策略

## 页面 5：模拟账户

### 5.1 顶部
- 日期选择器（可回溯历史快照）
- 4 个指标卡片：总资产 | 可用资金 | 持仓市值 | 今日盈亏

### 5.2 收益趋势图
- Plotly 面积图：x 轴=日期，y 轴=总资产
- 曲线下方填充绿色渐变
- 标注最高点和最低点

### 5.3 当前持仓表
- 表格：代码 | 名称 | 持仓数量 | 成本价 | 现价 | 市值 | 盈亏% | 盈亏金额
- 盈亏% 绿色（正）/ 红色（负）

### 5.4 今日成交
- 表格：时间 | 代码 | 方向 | 价格 | 数量 | 金额

## 交互规范

- 所有 Hover 弹出信息卡统一样式：深色半透明背景，白色文字，圆角 8px，阴影
- 页面切换无刷新（SPA）
- 分时图/收益曲线用 Plotly.js 交互图表（缩放、hover 数据点）
- 表格支持表头点击排序
- 加载状态：骨架屏（shimmer）而非转圈
- 空数据展示优雅（「暂无数据」占位）
- 移动端响应式：指标卡片 3 列变 2 列变 1 列，表格横向滚动
